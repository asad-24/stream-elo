# Admin And CMS Guide

## Current State

The public website and contact inquiry backend exist. A full admin dashboard, authentication, role-based permissions, CMS CRUD screens, and publishing controls are not implemented in this repository yet.

## Required Before This Guide Becomes Operational

- Login route and session handling
- Super Admin and Editor roles
- Password hashing
- Admin dashboard shell
- CRUD screens for projects, productions, people, BTS galleries, media, pages, settings, and contact inquiries
- Publish/unpublish workflow
- SEO metadata editor
- Audit logging

## Future Admin Workflows

- Log in through `/admin/login`.
- Change password on first login.
- Add films, documentaries, theatre productions, music projects, success stories, BTS galleries, and media.
- Link YouTube, MP4 CDN, or HLS video URLs.
- Edit homepage sections and global contact settings.
- Review and update contact inquiry statuses.
- Publish or unpublish content.
- Update SEO metadata per page/project.

This guide should be expanded after the admin/CMS implementation is completed.
